Github Procedures
Created 2020-05-20| Updated 2020-05-22
Word count:147

Step 1: 建立空文件夹

$ hexo -v

$ hexo init MHO

$ cd MHO

$ cnpm install

Step 2: 通过Github Desktop上传到github上，修改_config.yml中的网址

$ cnpm install hexo-deployer-git –save

Step 3: 下载themes: ayer, butterfly, next, and autumn

$ git clone -b master https://github.com/jerryc127/hexo-theme-butterfly.git themes/Butterfly

$ git clone https://github.com/Shen-Yu/hexo-theme-ayer.git themes/ayer

$ git clone https://github.com/FrontendSophie/hexo-theme-autumn.git themes/autumn

$ git clone https://github.com/theme-next/hexo-theme-next themes/next

$ git clone https://github.com/PhosphorW/hexo-theme-academia.git themes/Academia

Step 4: 编辑Butterfly的_config.yml

  1. npm install hexo-renderer-pug hexo-renderer-stylus –save
  2. 图片 & 文字；
  3. $ hexo new page tags $ hexo new page categories $ hexo new page link
